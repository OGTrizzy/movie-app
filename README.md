# 🎬 MovieFinder

A modern and beautiful movie & TV database built with React and TypeScript.

[Live Demo](https://movie-app-six-iota-87.vercel.app/) • [GitHub](https://github.com/OGTrizzy/movie-app)

## ✨ Features

- Search across **Movies** and **TV Shows**
- Browse **Popular**, **Top Rated**, and personalized **Watchlist**
- Detailed movie & TV pages with cast, genres, ratings and more
- Add/remove favorites with persistent storage
- Fully responsive design with smooth animations
- Dynamic theming based on content

## 🛠 Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS
- **State Management**: Zustand + TanStack Query
- **Routing**: React Router v6
- **API**: The Movie Database (TMDB)
- **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/OGTrizzy/movie-app.git
cd movie-app

# Install dependencies
npm install

# Start development server
npm run dev
